# Agentic Honeypot for Scam Detection & Intelligence Extraction

## Problem Statement
This project implements Agentic Honey-Pot for Scam Detection & Intelligence Extraction
as part of the India AI Impact Buildathon.

The system detects scam messages, autonomously engages scammers using a human-like AI agent,
extracts scam-related intelligence, and reports the final result to the GUVI evaluation endpoint.

---

## Features
- Scam intent detection
- Agentic multi-turn conversation handling
- Human-like responses without revealing detection
- Intelligence extraction:
  - UPI IDs
  - Phone numbers
  - Phishing links
  - Suspicious keywords
- Secure REST API
- Mandatory GUVI final callback implemented

---

## Tech Stack
- Python 3
- FastAPI
- Uvicorn
- Requests
- Regex-based intelligence extraction
